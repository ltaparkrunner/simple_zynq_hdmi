A simple implementation of the HDMI interface on the Z-turn Lite board from Myir on IPs, Verilog, System Verilog.

In a case the project is open badly in Vivado, execute in a Terminal:

 export LANG=en_US.UTF-8
 export LC_CTYPE="en_US.UTF-8"
 export LC_NUMERIC="en_US.UTF-8

Then start Vivado from the same Terminal:

/path_to_Vivado_folder/Vivado/2020.1/bin/vivado

Then open the project in Vivado.
